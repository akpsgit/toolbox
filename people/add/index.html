Add or Update People in the Toolbox

The API Specification Toolbox is a community project and runs 100% on Github so that anyone can submit a pull request and add or update any of the people listed here on the toolbox. You can find the people all listed in the _people folder in the repo, and each tool has a single YAML file stored as an .md file. Using YAML front end matter to publish the data to the listing and detail pages.

To update or add people to the site you can fork the repository for the project, add or update any of the files, and then submit a pull request with all the changes. If you don't feel comfortable submitting a pull request you can just submit an issue using the "new-people" label. If you are looking to help, we can really use assistance with some data entry, adding any of the existing people that have been submitted as a Github issue--we really appreciate the community contribution.
